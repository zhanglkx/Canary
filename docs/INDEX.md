## 🔍 如何使用

1. **新手入门**: 从 `README.md` 和 `QUICKSTART.md` 开始
2. **了解架构**: 查看 `architecture/` 目录下的文档
3. **开发指南**: 参考 `development/` 目录下的文档
4. **部署项目**: 按照 `deployment/` 目录下的步骤文档执行
5. **遇到问题**: 查看 `troubleshooting/` 目录下的解决方案
6. **电商功能**: 参考 `ecommerce/` 目录下的实现文档
